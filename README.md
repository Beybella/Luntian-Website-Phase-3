## üó∫Ô∏è Site Structure

The website is structured to be intuitive for online shoppers:

### Core Pages
* **`index.php`** ‚Äì **Homepage:** Features current promotions, seasonal highlights, and quick links.
* **`products.php`** ‚Äì **Product Catalog:** A dynamic display of all available flowers and arrangements fetched from the database.
* **`about.php`** ‚Äì **About Us:** The company story, mission, and florist values.
* **`contact.php`** ‚Äì **Contact Us:** Functional contact form, shop details, and social media links.
* **`season.php`** ‚Äì **Seasonal Blooms:** Highlights collections based on the current season (e.g., Valentine's Day).

### E-Commerce & User Flow
* **`cart.php`** ‚Äì **Shopping Cart:** Allows users to review items, adjust quantities, and manage their selection.
* **`checkout.php`** ‚Äì **Checkout:** Handles the final order processing and payment details.
* **`success.php`** ‚Äì **Order Confirmation:** Displayed after a successful transaction.
* **`signup.php` / `login.php`** ‚Äì **Authentication:** User registration and secure login system.
* **`profile.php` / `my_orders.php`** ‚Äì **User Dashboard:** Allows customers to manage their profile and view purchase history.

### Backend & Support
* **`includes/`** ‚Äì Contains reusable components:
    * `db.php`: Database connection settings.
    * `header.php` & `footer.php`: Global site navigation and footer.
* **`admin_orders.php`** ‚Äì **Admin Dashboard:** A private area for managing and fulfilling incoming orders.
* **`style.css`** ‚Äì Contains all the visual styling and layouts.
* **`script.js`** ‚Äì Handles client-side interactivity and animations.
* **`assets/`** ‚Äì Storage for all images, icons, and brand fonts.
